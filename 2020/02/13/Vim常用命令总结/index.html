Vim常用命令总结

毫无可读性的学习笔记

vimtutor记录

常用命令

x normal模式下删除

i 光标处插入文本

A 行末添加文本

a 光标后插入

R 一直替换知道按ESC退出

normal模式下:

dw 从光标处删除至下一个单词起始处

de 从光标处删除到单词末尾

d$ 从光标处删除至当前行尾

使用计数指定动作

e.g.:2w 3e 0(移动光标到行首)

使用计数以删除更多

e.g.:d2w

操作整行

e.g.:2dd

撤销类命令

u 撤销最后执行的命令

U 撤销对整行的修改

CTRL-R 撤销以前的撤销命令，恢复以前的操作结果

小结

Normal模式下修改命令格式：
operator [number] motion

置入类命令

dd删除整行，保存到VIM寄存器中，光标移动到准备置入位置的上方，Normal模式下p粘贴置入

替换类命令

输入r和一个字符替换光标所在位置字符

更改类命令

ce cw 改变文本直到一个单词的末尾

c$ 改变至行尾

定位及文件状态

CTRL-G 显示光标所在行位置和文件状态信息

[number]G 跳转到文件指定行

G 跳转到文件最后一行

gg 跳转到文件第一行

搜索类命令

/要查找字符串<回车> 从开头开始查找

?要查找字符串<回车> 从末尾逆向查找

n 查找同上一次的字符串

N 相反方向查找同上一次的字符串

CTRL-O 回到之前位置，重复按回退更多步

CTRL-I 跳转到最新位置

% 配对括号查找

替换命令

:s/old/new <回车> 将光标所在行第一个old替换成new

:s/old/new/g <回车> 将光标所在行所有old替换成new

:#,#s/old/new/g （其中#,#代表替换替换操作的若干行首尾两行行号）

:%s/old/new/g 替换整个文件种的每个匹配串

:%s/old/new/gc 会找到整个文件中每个匹配串，并且对每个匹配串提示是否进行替换

VIM执行外部命令

!:外部命令（包括带命令行参数的）

:r FILENAME 将文件提取到光标处

:r !ls 读取ls命令的输出到放置在光标下

打开类命令

o 光标下方打开新一行并进入插入模式

O 光标上方打开新一行并进入插入模式

复制粘贴文本

view模式下选择，y复制，p粘贴

y也可作操作符，如yw复制一个单词

设置类命令选项

:set ic (Ignore Case)忽略大小写

:set noic 取消忽略大小写

/匹配项\c <回车> 仅在一次查找时忽略字母大小写

:set hls (hlsearch) 匹配项高亮显示

:nohlsearch (移除匹配项高亮显示)

:set is (incsearch)查找短语时显示部分匹配

(ps:设置中选项名可使用完整版本，也可用缩略版本)

使用在线帮助系统

help cmd 找到关于cmd命令的帮助

CTRL-W CTRL-W 在窗口之前跳转

:q 关闭帮助窗口

创建vimrc

启动脚本文件保存偏好设置

补全功能

CTRL-D 查找可能的命令补全结果

TAB 使用一个补全

VIM 果真不是我等平凡人的大脑能掌握的编辑器 :)
